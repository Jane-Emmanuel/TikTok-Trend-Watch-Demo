###  What It Does

* 🎵 See the **Top 5 Trending Sounds**
* 🌍 Track **viral filters** & global hashtags
* 🤓 Fun TikTok **fact of the day**
* 🗳️ Vote for your favorite trend
* 📊 Live updates with `st.metric`

---

###  Built With

* Streamlit (Frontend)
* Python
* `pandas` + `requests`
* GitHub for version control
